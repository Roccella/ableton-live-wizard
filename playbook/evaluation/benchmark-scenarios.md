# Benchmark Scenarios

## Scenario Types
- `Loop already works but needs next steps`
- `Loop is missing one obvious role`
- `Loop has a clear genre but weak section contrast`
- `Loop sits between two nearby genres`
- `Loop is sparse and close to empty`

## Genre Order
- `House`: `M0.5` first, then all scenario types across `M1-M4`, then `M5`
- `Techno`: first three scenario types, milestones `M1-M3`
- `Trance`: first four scenario types, milestones `M1-M3`
- `Drum & Bass`: first two scenario types, milestones `M1-M2`

## Recording Rule
- Each run should be written into `research/playbook-validation/` using the benchmark template.
