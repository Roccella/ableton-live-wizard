# Protocol - Modify Loop

## Goal
- Apply or recommend one coherent musical change to an existing loop.

## Default Behavior
- Preserve genre and loop identity.
- Change one thing strongly rather than many things weakly.
- Prefer the smallest change that creates audible difference.

## Questions To Resolve
- Which role should change?
- Is the goal more energy, more contrast, more groove, more tension, or more clarity?
- Should the loop stay recognizably the same or become a new variant?

## Output Shape
- `Intended change`
- `Why this change fits`
- `Exact next action or prompt`
