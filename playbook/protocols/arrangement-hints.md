# Protocol - Arrangement Hints

## Goal
- Suggest plausible next sections from a loop or selected scene.

## Behavior Rules
- Infer whether the current material feels like intro, groove, break, lift, or peak.
- Suggest the next section by function, not by generic template order alone.
- Keep suggestions compatible with Live workflows that start from loops and scenes.

## Output Shape
- `Current section feeling`
- `Most likely next section`
- `Alternative next section`
- `One key musical change per option`
