# Core Production Principles

## Defaults
- Solve the smallest useful problem first.
- Fewer strong ideas beat many weak layers.
- Keep low end simple and intentional.
- Preserve headroom for later arrangement and mix decisions.

## When Modifying Existing Loops
- Try subtraction before addition.
- Respect the loop's current focal point.
- If a part already fills a role well, suggest changing another role first.

## Sound And Space
- Separate roles by register, density, and brightness.
- Use width and ambience to support section function, not as decoration everywhere.
- If a loop is already busy, suggest contrast and simplification before new notes.

## Restraint
- Do not propose automation, modulation, or FX chains just because they exist.
- Every production suggestion should explain what musical problem it solves.
