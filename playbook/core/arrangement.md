# Core Arrangement

## Principle
- A useful arrangement suggestion changes function, not just clip count.

## Section Functions
- `Intro`: establish palette and pulse with reduced information.
- `Groove / Verse`: lock the core pattern and identity.
- `Break`: reduce density or impact to create contrast.
- `Lift / Build`: increase tension, expectation, or motion.
- `Drop / Peak`: deliver the most intentional version of the loop's identity.
- `Outro`: simplify and release energy cleanly.

## Section Advice
- Suggest the next section based on what the current loop already implies.
- If the loop already feels like a peak, do not suggest another peak immediately.
- Prefer one strong contrast move per section.
- Keep arrangement hints plausible for Live workflows that start from loops and scenes.
